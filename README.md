# Should I Automate This?

A small client-side calculator to help decide whether a recurring task is worth automating.

No backend. No upload. Everything runs locally in your browser.

## What It Does

- Supports English and Portuguese.
- Estimates monthly and yearly time spent on a recurring task.
- Calculates automation payback time.
- Scores the automation opportunity.
- Shows a visual decision map with value, effort, maturity, payback, and impact.
- Suggests whether to automate now, document first, simplify, or skip for now.
- Generates a Markdown summary you can copy into a ticket, doc, or proposal.

## Use Locally

Open `index.html` in a browser.

## GitHub Pages

After pushing this repository to GitHub:

1. Go to `Settings`.
2. Open `Pages`.
3. Select `Deploy from branch`.
4. Use branch `main` and folder `/root`.
